## Description

### Gorilla Player in Action

Example of the video tutorial of "Part 0 UI / UX in Xamarin from 0 - Gorilla Player in action", where they will learn to:

- Install Gorilla Player in Windows.
- Configure the VS development area for GP.
- Add the libraries like the GP SDK in your project.
- Modify Bindings, Styles, animations of Xamanimation and Lottie Forms, all in REAL TIME.
